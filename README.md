# Deal Intelligence Lab

A portfolio of investment banking models, transaction analysis and workflow automation projects covering mergers and acquisitions, equity capital markets, leveraged buyouts, treasury risk management and derivatives.

Developed as part of my preparation for M&A Investment Banking recruiting and focused on improving analyst productivity through financial modeling and automation.

---

## Projects

### AI-Enabled M&A Deal Automation Platform
Automated SEC filing extraction and review-ready merger model generation validated across public M&A transactions.

### SpaceX Post-IPO Valuation & Aftermarket Analysis
DCF, SOTP and IPO pricing analysis with post-offering performance attribution.

### SpaceX-Cursor Acquisition Analysis
Strategic M&A analysis, exchange ratio modeling and transaction valuation.

### BlackLine Take-Private LBO
Five-year LBO model including debt schedules, returns analysis and value creation bridge.

### Treasury Hedging Workbook
DV01-neutral treasury futures hedge simulation with automated rebalancing.

### Simulated S&P 500 Options Portfolio
Greeks-based options strategy with systematic risk controls.
