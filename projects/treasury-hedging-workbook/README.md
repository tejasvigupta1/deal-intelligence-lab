# Automated Treasury Hedging Workbook

## Objective

Develop an automated treasury hedging framework using Treasury futures to manage duration risk and interest rate exposure.

---

## Investment Question

How can a portfolio manager minimize interest rate risk while maintaining target portfolio exposure?

---

## Framework

The model incorporates:

- DV01 calculations
- Modified Duration analysis
- Treasury futures hedge ratios
- Contract sizing
- Margin tracking
- Hedge rebalancing

---

## Risk Metrics

The workbook tracks:

- Portfolio DV01
- Hedge DV01
- Net Duration Exposure
- Tracking Error
- Hedge Effectiveness

---

## Automation Features

- Automated hedge sizing
- Dynamic rebalancing
- P&L tracking
- Margin monitoring
- Risk reporting
